Introduction

In this tutorial, we will cover the fundamentals of performing an NCA analysis with PuMaS of an example dataset in which a single oral dose was administered and the concentration of two analytes (parent and metabolite) was measured.

The dataset

A single oral dose of 2000 mg was administered to 24 different subjects. Samples were collected every 30 minutes.

Let's start reading the dataset. By using the missingstring option we are specifying how the missing values are labeled in our dataset.

data = CSV.read("./data/SD_oral_2analytes.csv",missingstring="NA")

Please, note that the DOSE column must only contain a value at the time the dose was administered.

Defining the units

Next we can define time, concentration and dose units so the report includes the units for the pharmacokinetic parameters.

timeu = u"hr"
concu = u"mg/L"
amtu  = u"mg"

mg

Defining the population object

Using the parse_ncadata function, the next step would be to define the population that we are going to use for the NCA. Within this function we need to specify the dataset, the name of the column with the subject identifier (id=), name of the time column (time=), name of the concentration column (conc=), name of the dose column (amt=), the interdose interval (ii=) multiplied by the time units, the name of the formulation column (formulation=) and the route of administration (route=).

In this case, since parent and metabolite concentrations were measured, we need to specify the grouping variable so that the PK parameters are calculated for both analytes (group=).

pop = parse_ncadata(data, id=:ID, time=:time, conc=:DV, amt=:DOSE, ii=24timeu, group=:Analyte,
    formulation=:Formulation, route=(ev="oral",), timeu=timeu, concu=concu, amtu=amtu)

Please, note that in the route=(ev="oral",) option the name between quotes should match the contents of the formulation column in our dataset.

Also note that in the function above by default the lower limit of quantification (LLQ) is 0 and concentrations that are below LLQ (BLQ) are dropped.

Let's say we want to specify a LLQ value of 0.4 mg/L, then we need to add llq=0.4concu to the function above:

pop = parse_ncadata(data, id=:ID, time=:time, conc=:DV, amt=:DOSE, ii=24timeu, group=:Analyte,
